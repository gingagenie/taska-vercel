# Taska Quotes/Invoices — Replit Quickstart


**Dev run**
1. Click **Shell** → run `npm install`
2. Click **Run** (or `npm run dev`)
3. Open the webview on port 5173. The API runs on 3000 and is proxied by Vite.


**Build & Prod**
```bash
npm run build
npm start
```
Express will serve the built app from `dist/`.


**Where to plug your real stuff**
- API: `server/index.js` → `/api/send` (email/PDF/Stripe/Xero)
- Presets: swap the in-memory `PRESETS` for your DB or Supabase
- Styling: Tailwind in `src/index.css`
```



Legend. I’ve turned it into a Replit-ready app on the canvas: React (Vite) + Tailwind front end, tiny Express API, dev proxy, and a “Send” endpoint that actually receives the document.

Quick start:

In Replit: make a new Node.js repl.

Copy the files from the canvas (keep the same paths).

In the Shell: npm install

Hit Run (or npm run dev). Open the webview (port 5173).
